aplikacja działająca na/w stronie ng-app

Zadeklarowanie nazwy aplikacji AngularaJS. 

index.html:
    <html ng-app="nazwaAplikacjiApp">

app.js:
    app = angular.module('nazwaAplikacjiApp', [wykorzystywane moduły, np. 'ngRoute']);

kontroler - funkcja kontrolujacą działanie wybranej części strony ng-controller

Zadeklarowanie nazwy funkcji kontrolującą działanie w ramach elementu (body, div, itp.), kontroluje kontroluje zmienne wykorzystywane w ng-model, czy {{...}}.
Deklaruje również funkcje wykorzystywane do obsługi zdarzeń, np w ng-click.
Zmienne dostępne są przez obiekt $scope. 

index.html:
    <element ng-controller="nazwaKontroleraCtrl" ...

ctrls.js:
    app.controller('nazwaKontroleraCtrl',
                   [wykorzystywane serwisy, np. $scope, '$http', ...,
                    function ($scope, $http, $location, $interval, $timeout) {
      :
    }]);

nowe komponenty

Zdefiniowanie komponentu nowyKomponent do wykorzystania na stronie w formie <nowy-komponent></nowy-komponent>, którego wygląd definiuje template, a "zachowanie" funkcja controler. 

index.html:
    <nowy-komponent></nowy-komponent>
app.js:
    angular.module('nazwaAplikacjiApp', ['nowyKomponent', ...]);
nowy-komponent.component.js:
    angular.
      module('nowyKomponent').
      component('nowyKomponent', {
        templateUrl: 'nowy-komponent.template.html',
        controller: ['$http', function nowyKomponentController($http) {
          var self = this;//to się bardzo przydaje w funkcjach
          self.nazwaPolaJakas = 'coś tam';
        }]
      });
nowy-komponent.template.html:
    

      {{ctrl.nazwaPolaJakas}}

pola, zmienne ng-model, ng-init, {{...}}, ng-bind, $scope

Związanie pola ze zmienną, nadanie wartości początkowej, pokazanie jej wartości. Zmienne są dostępne w kontrolerze przypisanym do części strony zawartej w elemencie. 

index.html:
    <input ng-model="nazwaZmiennej" ng-init="nazwaZmiennej=jakaś nazwa pola" ...
    ... {{nazwaZmiennej}} ...
    ... {{wyrażenie}} ...
    <span ng-bind="nazwaZmiennej"></span>

module.js:
    $scope.nazwaZmiennej = ...;
  • Proszę wpisać coś do pola nazwaPola , zawierajacego ng-model
  • pole nazwaPola ma wartość = "{{nazwaPola}}", wyliczoną w {{}}
  • pole nazwaPola można potraktować filtrem np. | uppercase = "{{nazwaPola | uppercase}}"
  • pole nazwaPola można ng-bind w spanie:
  • pola można zainicjowane w nazwaKontroleraCtrl lub ng-init i uzyskać nazwaPola1 = "{{nazwaPola1}}", nazwaPola2 = "{{nazwaPola2}}" oraz nazwaPola1,nazwaPola2 = "{{nazwaPola1+","+nazwaPola2}}"
